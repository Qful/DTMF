## [简介](https://github.com/lite-life/elite/wiki) 

elite是litelife的一个重要组成部分，通过elite项目向大家介绍定制化的个人数字化产品，通过拆分集成的思路，提供小众化的办公设备。

项目主要针对目标包括效率集成和优化，例如功能丰富的键鼠外设集成更多自动化能力，各种功能目的的电脑扩展板卡和工具设备等。
